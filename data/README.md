# Data Layer

## Overview
This folder contains data used in the pipeline.

## Structure

- raw/ - source data (planned, actual, employees)
- processed/ - transformed dataset for analytics

## Raw Data

### planned.csv
Planned workload allocation per employee and project.

### actual.csv
Actual timesheets data.

### employees.csv
Employee master data (unit, fte, roles).

## Processes Data

### workload.csv
Final dataset for BI and reporting.
